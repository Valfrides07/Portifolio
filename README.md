# Portfólio - (Portfólio Teste)

- Observação: Este portfólio foi feito com intuito de teste, não é nunca foi feito com o objetivo de ser oficial!

Este repositório contém o código-fonte do meu portfólio teste, desenvolvido com foco em aprendizado prático, organização de código e evolução contínua como desenvolvedor.

- Visão Geral

O portfólio apresenta:

Uma seção de apresentação pessoal

Um resumo da minha trajetória e objetivos na área de tecnologia

Uma vitrine de projetos desenvolvidos para estudo e experimentação

Animações suaves acionadas conforme o usuário navega pela página

Todo o layout foi pensado para ser simples, direto e funcional, priorizando clareza visual e leitura confortável.

- Tecnologias Utilizadas:

HTML5
Estrutura semântica e organizada da aplicação.

CSS3
Estilização completa do layout, responsividade e animações visuais.
Uso de Flexbox, media queries e efeitos de transição.

JavaScript (Vanilla)
Interatividade e animações baseadas em scroll utilizando a API IntersectionObserver.

Google Fonts
Tipografia personalizada para melhorar a identidade visual.

- Funcionalidades em Destaque

Layout responsivo para desktop e dispositivos móveis

Animação progressiva dos cards de projetos ao entrar no viewport

Estrutura modular e fácil de manter

Código comentado com foco em aprendizado e clareza

- Projetos Apresentados

O portfólio inclui projetos desenvolvidos com foco em estudo e aprimoramento técnico, como:

Gerador de Senhas Customizáveis
Aplicação que cria senhas seguras a partir de critérios definidos pelo usuário.

Sistema de Cardápio Digital com CMS
Plataforma com visualização pública e área administrativa para gerenciamento de produtos.

Landing Page Interativa
Página focada em experiência do usuário, com narrativa visual e interatividade.
